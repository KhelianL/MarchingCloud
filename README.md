# MarchingCloud : Point cloud rendering
This project aims to offer a program capable of rendering point clouds using sphere tracing.  
This is a university project on which the students will be graded.

## Dependencies
- C++ : v17.0  
- CUDA : v10.1 
- Qt : v5.7

## Execute Commands
```console
qmake
make -j

```
